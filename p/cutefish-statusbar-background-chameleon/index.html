Cutefish 菜单栏背景颜色识别方案

Jul 25, 2022
1 minute read

macOS Big Sur 版本中顶部菜单栏引入了一个新特性，就是菜单栏背景颜色与桌面壁纸适应效果，相比之前更加融合。

从实际效果中看不仅仅是将背景透明+模糊，它会根据壁纸图片顶部区域中提取主题颜色。

实现方案

平均值法

将图像数据中所有的 rgb 值加起来，再除以图像的面积，求平均值。

QImage img("fileName");
QSize size = img.size();

long long sumR = 0, sumG = 0, sumB = 0;
// 面积
int measureArea = size.width() * size.height();

// 遍历图片每一个坐标
for (int y = 0; y < size.height(); ++y) {
    QRgb *line = (QRgb *)img.scanLine(y);
    
    for (int x = 0; x < size.width(); ++x) {
        sumR += qRed(line[x]);
        sumG += qGreen(line[x]);
        sumB += qBlue(line[x]);
    }
}

sumR /= measureArea;
sumG /= measureArea;
sumB /= measureArea;

QColor themeColor = QColor(sumR, sumG, sumB);
