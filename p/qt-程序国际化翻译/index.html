Qt 程序国际化翻译

Apr 02, 2021
1 minute read

Qt 提供了支持 C++ 和 QML 的翻译工具，使用 lupdate 工具同步源代码，使用 lrelease 工具生成应用程序创建运行时使用的翻译文件。 由于项目统一使用 CMake 工具来编译源代码，所以下面的教程使用 CMake 来讲解。 需要注意的是 C++中使用 QObject::tr("") 的函数，QML 里使用 qStr("")

首先生成对应的 TS 文件（比如中文）：

1

lupdate ./src -ts ./translations/zh_CN.ts

如果项目的源代码里有些需要翻译的字符串删除了，可以在后面加入一个参数：

1

lupdate ./src -ts ./translations/zh_CN.ts -no-obsolete

它的作用就是删除所有过时和消失的字符串。

生成 TS 文件后可以用 Qt Linguist 来进行编辑翻译。

然后在 CMakeLists.txt 中加上下边代码片段：

1
2
3
4
5

file(GLOB TS_FILES translations/*.ts)
qt5_create_translation(QM_FILES ${TS_FILES})
add_custom_target(translations DEPENDS ${QM_FILES} SOURCES ${TS_FILES})
add_dependencies(${PROJECT_NAME} translations)
install(FILES ${QM_FILES} DESTINATION /usr/share/${PROJECT_NAME}/translations)

    QTranslator *translator = new QTranslator(QGuiApplication::instance());
    if (translator->load(qmFilePath)) {
