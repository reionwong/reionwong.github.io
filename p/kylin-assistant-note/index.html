kylin-assistant 项目源码阅读笔记

Aug 18, 2019
3 minute read

根据官方介绍，麒麟助手是优客团队推出的一款开源的 Linux 系统管理软件，将 Linux 系统命令行方式的系统管理操作方式转化为更易用的可视化操作。麒麟助手集清理加速、健康优化、个性设置以及功能大全于一体，是用户便捷又贴心的系统助手。

闲来无事就看了看这个项目，源码地址：https://github.com/eightplus/kylin-assistant

这个项目主要使用 C++、Python 进行开发，前端图形库使用强大的 Qt，后端使用 Python，前端与后端的通信是借助于 D-Bus，本文主要对后端代码进行剖析，看看它是如何获取系统信息。

当然这些文件夹下包含各种子文件夹，以上内容仅概述，下面是各个后端模块的讲解。

MonitorBall

这个模块包含了清理内存，获取 cpu 占用百分比，获取内存占用等信息；该模块引入了 psutil、subprocess 库，psutil 是一个跨平台库，能够轻松实现获取系统运行的进程和系统利用率（包括CPU、内存、磁盘、网络等）信息，它主要用来做系统监控，性能分析，进程管理。subprocess 允许你去创建一个新的进程让其执行另外的程序，并与它进行通信，获取标准的输入、标准输出、标准错误以及返回码等。

根据 cleanup_memory 函数名可知，它是一个可以清理内存功能，使用 subprocess 调用 shell 程序。

# clean up memory
def cleanup_memory(self):
    '''force changed to disk, update the super block'''
    pone = subprocess.Popen(["sync"], shell=True)
    pone.wait()

    '''drop pagecache, dentries and inodes to free the memory'''
    ptwo = subprocess.Popen(["echo 3 > /proc/sys/vm/drop_caches"], shell=True)
    ptwo.wait()

首先它调用了 sync，这个命令来进行数据的写入动作，然后 pagecache、dentries、inodes 释放内存。

def get_total_memory(self, symbol = "m"):
    baseData = psutil.virtual_memory()
    value = baseData.total
    return self.bytes2human(value, symbol)

首先它调用 psutil 里的函数，存入相关的变量，然后再进行数据转换(bytes to human)。

  • 得到 used：baseData.total - baseData.available
  • 得到 free: baseData.available
  • 得到百分比：baseData.percent

# get network total flow, return (up, down)
def get_network_flow_total(self, symbol = "k"):
    network = psutil.network_io_counters()
    sent = network.bytes_sent
    recv = network.bytes_recv

    if(symbol == "b"):
        return (sent, recv)
    elif(symbol == "k"):
        k_up = float(sent) / 1024
        ups = str(k_up)
        ups = ups[0:ups.find(".") + 2]

        k_down = float(recv) / 1024
        downs = str(k_down)
        downs = downs[0:downs.find(".") + 2]

        return (ups, downs)
    else:
        return None

def bytes2human(self, value, symbol):
    if symbol == "k":
        value = value / 1024
    elif symbol == "m":
        value = value / 1024 / 1024
    elif symbol == "g":
        valuef = float(value) / 1024 / 1024 / 1024
        values = str(valuef)
        value = values[0:values.find(".") + 2]
    else:
        pass
    return str(value)

RELEASEPATH = "/etc/ubuntukylin-release"

def get_distro(self):
    '''It should be: DISTRIB_DESCRIPTION="UbuntuKylin 13.10'''
    #FILEPATH  RELEASEPATH
    distro = ""
    if os.path.exists(RELEASEPATH):
        with open(RELEASEPATH, "r") as fsys:
            for line in fsys:
                if line.startswith("DISTRIB_DESCRIPTION"):
                    tmp = line
                    break
        # kobe: remove '"' and '\n'
        front = tmp.split('=')[1].replace('"', '').replace('\n', '') #(LP: #1240862)
        distro = front + '-' + platform.dist()[2]
    elif os.path.exists("/etc/os-release"):
        with open("/etc/os-release", "r") as fsys:
            for line in fsys:
                if line.startswith("PRETTY_NAME"):
                    tmp = line
                    break
        distro = tmp.split('=')[1].replace('"', '').replace('\n', '')
    else:
        a = platform.dist()[0]
        b = platform.dist()[1]
        c = platform.dist()[2]
        distro = '-'.join((a,b,c))
        
    return distro

首先检查 /etc/ubuntukylin-release 是否存在，如果存在就对每一行进行解析，如遇到 DISTRIB_DESCRIPTION 字段则截取无关字符串最后得到结果。

如果不存在上面的 ubuntukylin-release 文件(这是 ubuntukylin 专有的 release 文件)，则判断 /etc/os-release 文件，这是众多发行版的统一标准，然后逻辑与上面说明一致。
