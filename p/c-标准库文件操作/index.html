C++ 标准库文件操作

Oct 06, 2018
2 minute read

在编程中，对一些文件读写操作是很有必要的；最近对 C++ 标准库挺感兴趣的。

文件流操作

在标准库中可以通过 fstream 来进行文件读写操作，包含了常用的文件操作，在 C++ 所有文件操作都是以 流(stream) 的方式进行的，fstream 也就是 文件流(file stream) 的意思；计算机并不区分二进制文件和文本文件，所有文件都是以二进制形式存储。

fstream 子类是通过 stream 来实现的，毫无疑问，ofstream、ifstream 的意思分别是输出文件流和输入文件流。

在 stream 这个类中有两个重要的运算符：

  • 流输出(«)，向流输出数据。
  • 流输入(»)，从流中输入数据。

#include <iostream>
#include <fstream>

int main(int argc, char *argv[])
{
    std::fstream in;
    in.open("/home/rekols/Desktop/test.txt", std::ios::out);

    if (!in.is_open()) {
        std::cout << "cannot open." << std::endl;
    }

    std::string content = "hello world";
    content.push_back('\n');
    in.write(content.c_str(), content.size());
    in.close();

    return 0;
}

#include <iostream>
#include <fstream>

int main(int argc, char *argv[])
{
    std::fstream file;
    file.open("/home/rekols/Desktop/test.txt", std::ios::in);

    if (!file.is_open()) {
        std::cout << "cannot open." << std::endl;
    }

    // 设置读位置到末尾
    file.seekg(0, std::ios::end);
    // 总长度
    int len = file.tellg();

    char *buffer = new char[len];
    // 回到开始位置
    file.seekg(0, std::ios::beg);
    // 整个文件读入 buffer
    file.read(buffer, len);
    file.close();

    std::cout << buffer;

    return 0;
}

is_open() 是检测文件是否打开成功，open() 成员函数的定义：

1
2

filebuf* open (const char* filename,  ios_base::openmode mode);
filebuf* open (const string filename, ios_base::openmode mode);

第一个参数是要打开的文件路径，第二个参数是要打开的模式，在 ios_base::openmode 定义了几个常量：

常量完整名介绍
ios_base::ininput文件读打开模式，支持输入操作
ios_base::outoutput文件写打开模式，支持输出操作
ios_base::binarybinary二进制打开模式，而不是文本
ios_base::ateat end打开后立即寻位到流结尾
ios_base::appappend每次写入寻位到流结尾
ios_base::trunctruncate在打开时舍弃流的内容

删除文件

可以直接 std::remove() 来进行文件删除操作；需要注意的是和一对送代器函数模板名字是一样的。

函数定义：int remove( const char* fname );，删除成功时返回值为 0，错误时为非零值。
